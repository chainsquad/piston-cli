************
Installation
************

Install with `pip`:
-------------------

.. code-block:: sh

   pip3 install piston-cli

Manual installation:
--------------------

.. code-block:: sh

   git clone https://github.com/xeroc/piston-cli
   cd piston-cli
   python3 setup.py install --user

Upgrade
-------

.. code-block:: sh

   pip install --user --upgrade piston-cli
